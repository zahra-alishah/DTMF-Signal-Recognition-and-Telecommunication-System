# DTMF Signal Recognition and Telecommunication System

This project implements a complete DTMF (Dual-Tone Multi-Frequency) communication system in MATLAB. The system generates DTMF signals, transmits them through a simulated communication channel using AM/DSB-SC modulation, performs coherent demodulation and noise filtering, and automatically detects transmitted digits through FFT-based frequency analysis.

## Features

* DTMF signal generation
* AM and DSB-SC modulation
* Coherent demodulation
* Noise filtering and signal recovery
* FFT-based frequency analysis
* Automatic DTMF digit detection
* End-to-end telecommunication system simulation

## Results

Successfully recovered transmitted digit sequences with high accuracy under noisy channel conditions, demonstrating robust signal detection and reliable communication system performance.
